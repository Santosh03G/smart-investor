Tech Stack
Layer Technology Purpose
Core Engine Rust + Tokio Zero-GC async runtime. WebSocket multiplexing, OHLCV aggregation, order routing.
Desktop Shell Tauri v2 OS native webview. ~10MB binary vs ~150MB Electron. IPC bridge between Rust and React.
Frontend React 18 + TypeScript + Vite Interactive workspace. Zustand for market data state. TailwindCSS styling.
Charts TradingView Lightweight Charts v4 Canvas/WebGL rendering. Handles thousands of candles without DOM jank.
Data Grid AG Grid Community Virtualized rows for order book depth, positions, screener output.
Persistence SQLite via sqlx AES-256-GCM encrypted API keys. OHLCV history. Chart layouts. Watchlists.
Local AI Ollama (Llama 3 / Mistral) Fully local LLM. No API credits. Earnings parsing, sentiment, trade journal analysis.
// setup
Quick Start
1 — Prerequisites 
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install Node 18+  →  https://nodejs.org

# Install Tauri CLI
npm install -g @tauri-apps/cli

# Install Ollama (for local AI)  →  https://ollama.com
ollama pull llama3
2 — Clone & install 
git clone https://github.com/your-username/native-terminal
cd native-terminal
npm install
3 — Dev server 
cargo tauri dev
# Opens a 1440×900 native desktop window
# Hot-reload on React changes · Rust recompiles on save
4 — Production build 
cargo tauri build
# Outputs: .exe (Windows) · .dmg (macOS) · .AppImage (Linux)
# Located at: src-tauri/target/release/bundle/

// security model
Privacy by Design
🔒
Keys never leave
Broker API keys are AES-256-GCM encrypted and machine-bound. They exist only in your local SQLite database.
🌐
No cloud backend
The app connects only to exchange WebSockets and your broker's official API endpoints. Zero intermediary servers.
🧠
Local AI only
All LLM inference runs via Ollama on your own GPU. Your trade data and market queries never reach a third-party AI API.
